== Description ==

PayJoe hilft Ihnen bei Ihrer Buchhaltung und gleicht Ihre Zahlungen von z.B. AMAZON, PayPal, Klarna, Billsafe usw. mit den Rechnungen ab.

Kostenfreier Testzugang unter <a href="http://www.payjoe.de" target="_blank">www.payjoe.de</a>

**Funktionen**

* automatischer, zeitgesteuerter Upload der Belege zu PayJoe

* Manueller Sofortupload der Belege

* Support für WooCommerce Germanized

* Support für WP Overnight WooCommerce PDF Invoices & Packing Slips

* Support für German Market

<hr>
Wordpress Plugin Entwicklung durch:  <a href="https://weslink.de" target="_blank">WESLINK</a>


== Installation ==

Installieren Sie dieses Plugin und tragen Sie die API Daten von PayJoe ein.

**Voraussetzungen**

* https://wordpress.org/plugins/woocommerce/

= Wie erhalte ich Support =

Schreiben Sie eine E-Mail an info@payjoe.de oder [erstellen Sie ein Ticket](https://hilfe.payjoe.de/online-haendler/Content/topics_payjoe/support/ticket_erstellen.htm) aus PayJoe heraus.

= Das Plugin löst Ihr Problem nicht? =

Das Plugin erfüllt nicht Ihre Anforderungen? Wir [programmieren Ihr individuelles WordPress Plugin](https://weslink.de) für Sie.
